# Data Cleaning & EDA Projects

This repository contains a collection of real-world datasets used to practice:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Statistical Analysis
* Pandas & NumPy Workflows

The goal is to move beyond exercises and develop practical skills by working with real datasets and extracting meaningful insights.

## Project Roadmap

### Category 1

* Students Performance Dataset
* IBM HR Analytics Employee Attrition

### Category 2

* Netflix Movies & TV Shows
* Superstore Sales Dataset

### Category 3

* California Housing Prices
* House Prices (Ames Housing)
* Insurance Claims / Medical Cost Personal Dataset

## Tools

* Python
* Pandas
* NumPy
* Matplotlib

## Objective

For each dataset:

1. Understand the data
2. Clean the data
3. Perform EDA
4. Create visualizations
5. Extract insights
6. Document findings

This repository serves as a record of my journey from learning Pandas to working with real-world data and preparing for machine learning projects.
